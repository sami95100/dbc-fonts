```
products/
  apple/
    iphone-16-pro.jpg

    iphone-16-pro-max.jpg
    iphone-15-pro.jpg
    iphone-15.jpg
    iphone-14-pro.jpg

  samsung/
    galaxy-s24-ultra.jpg

    galaxy-s23-ultra.jpg
  google/
    pixel-9-pro.jpg
    pixel-9.jpg

```

## Brands
Put brand logos here:

```
brands/
  apple.png
  samsung.png
  google.png
  xiaomi.png
```

## Naming Convention
- Use lowercase
- Use hyphens (not spaces)
- Format: `model-name.jpg` (e.g., `iphone-16-pro.jpg`)
